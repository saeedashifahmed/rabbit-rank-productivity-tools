# Rabbit Rank Productivity Tools

A modern collection of free, browser-based productivity utilities built with Next.js, React, TypeScript, and Tailwind CSS.

## About

Rabbit Rank Productivity Tools includes practical tools for writers, developers, marketers, and SEO professionals.

This toolset is developed by Rabbit Builds (https://rabbitbuilds.com/), owned and operated by Rabbit Rank LLC (https://rabbitrank.com/).

## Included Tools

- Word Counter
- Case Converter
- URL Trimmer
- Image Compressor
- HEIC to JPG Converter
- SVG Compressor

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Radix UI components

## Getting Started

### Prerequisites

- Node.js 18.18+ (recommended: latest LTS)
- npm (or pnpm)

### Installation

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Scripts

- `npm run dev` - Start local development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run lint checks
